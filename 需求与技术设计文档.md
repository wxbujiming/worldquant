# WorldQuant Brain Web 界面 — 需求与技术设计文档

## 1. 项目背景

基于 `wqb` Python 库（WorldQuant Brain API 的 Python 封装），构建一个本地运行的 Web 可视化界面，方便通过浏览器操作 WB 平台的核心功能，替代命令行/Python 脚本的交互方式。

## 2. 技术栈

| 层级 | 技术选型 |
|---|---|
| 后端框架 | FastAPI (Python 3.13) |
| 前端框架 | Vue 3 + Vite |
| 前端 UI 库 | Naive UI |
| 状态管理 | Pinia |
| 前端路由 | Vue Router |
| HTTP 客户端 | Axios |
| 实时通信 | WebSocket（模拟进度推送） |
| 运行方式 | Uvicorn 本地启动，前后端同端口部署 |

## 3. 功能需求

### 3.1 认证管理

| 功能 | 说明 |
|---|---|
| 登录 | 输入 WB 邮箱和密码，通过 `WQBSession` 认证 |
| 登录状态展示 | 显示当前登录用户，session 自动续期 |
| 登出 | 清除 session |

### 3.2 数据集浏览

| 功能 | 说明 |
|---|---|
| 搜索数据集 | 按 Region、Delay、Universe、分类、关键词等筛选 |
| 数据集详情 | 查看数据集覆盖范围、评分、关联 Alpha 数等 |
| 数据字段列表 | 查看数据集下所有字段 |
| 字段详情 | 字段类型、分类、使用统计等 |

### 3.3 算子浏览器

| 功能 | 说明 |
|---|---|
| 算子列表 | 展示所有可用算子 |
| 算子搜索 | 按名称/类别搜索过滤 |

### 3.4 Alpha 管理

| 功能 | 说明 |
|---|---|
| Alpha 列表 | 按状态、Sharpe、Region、Delay、Universe 等多维度筛选 |
| Alpha 详情 | 查看代码、设置、模拟结果、回测指标 |
| 创建/编辑 Alpha | 在线编辑 Alpha 公式代码 |
| 属性修改 | 修改名称、标签、颜色、收藏、隐藏等 |

### 3.5 Alpha 模拟

| 功能 | 说明 |
|---|---|
| 提交模拟 | 将 Alpha 代码提交到 WB 模拟 |
| 进度追踪 | 实时推送模拟进度（WebSocket） |
| 结果展示 | Sharpe、收益率、PnL、换手率、回撤、边际等指标可视化 |
| 批量模拟 | 并发提交多个 Alpha |

### 3.6 Alpha 提交

| 功能 | 说明 |
|---|---|
| 提交比赛 | 将 Alpha 提交到指定比赛 |
| 状态检查 | 检查 Alpha 编译/校验状态 |

### 3.7 Alpha 公式辅助生成（扩展功能）

| 功能 | 说明 |
|---|---|
| 模板生成 | 基于选定的数据集/字段/算子，提供常用公式模板 |
| 公式组装 | 通过图形化选择算子、字段、参数来组装 Alpha 公式 |

## 4. 后端 API 设计

### 4.1 认证

```
POST   /api/auth/login          # 登录，body: {email, password}
DELETE /api/auth/logout         # 登出
GET    /api/auth/status         # 获取认证状态
```

### 4.2 数据集

```
GET    /api/datasets            # 搜索数据集（query: region, delay, universe, ...）
GET    /api/datasets/{id}       # 获取数据集详情
```

### 4.3 数据字段

```
GET    /api/fields              # 搜索字段（query: dataset_id, region, delay, universe, ...）
GET    /api/fields/{id}         # 获取字段详情
```

### 4.4 算子

```
GET    /api/operators           # 搜索算子（query: search, ...）
```

### 4.5 Alpha

```
GET    /api/alphas              # 筛选 Alpha（支持 30+ 过滤参数）
GET    /api/alphas/{id}         # 获取 Alpha 详情
PATCH  /api/alphas/{id}         # 修改 Alpha 属性
POST   /api/alphas/{id}/simulate   # 提交模拟（返回 task_id）
GET    /api/alphas/{id}/simulate/{task_id}/status  # 查询模拟状态
POST   /api/alphas              # 创建新 Alpha
PUT    /api/alphas/{id}/code    # 更新 Alpha 代码
```

### 4.6 模拟与提交

```
POST   /api/simulate            # 模拟一段 Alpha 代码
GET    /api/simulate/{task_id}  # 获取模拟结果
POST   /api/alphas/{id}/check   # 检查 Alpha 状态
POST   /api/alphas/{id}/submit  # 提交到比赛
WS     /ws/simulate/{task_id}   # WebSocket 推送模拟进度
```

## 5. 前端页面与路由

| 路由 | 页面 | 说明 |
|---|---|---|

| `/datasets/:id` | 数据集详情 | 查看字段列表 |
| `/operators` | 算子浏览器 | 浏览搜索算子 |
| `/alphas` | Alpha 列表 | 筛选管理 Alpha |
| `/alphas/:id` | Alpha 详情 | 查看代码、结果、属性编辑 |
| `/alphas/:id/edit` | Alpha 编辑 | 在线编辑公式 |
| `/simulate` | 模拟控制台 | 提交模拟、查看进度和结果 |

## 7. 数据流

```
浏览器 (Vue)
    ↕ Axios HTTP / WebSocket
FastAPI 后端
    ↕ WQBSession (requests)
WorldQuant Brain REST API
```

关键设计点：

- **Session 管理**：WQBSession 实例在后端内存中维护，每个登录用户对应一个 session
- **模拟异步化**：模拟耗时较长，后端提交后通过 async/await + WebSocket 推送进度
- **自动重试**：利用 WQBSession 内置的重试机制处理 WB API 限流
